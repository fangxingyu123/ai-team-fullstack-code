## 🔧 Bug 修复详情

### 主要 Bug：打到后面无法加分

**发现时间:** 2026-03-05 21:01  
**报告人:** 老板  
**根本原因:** 连击计时器逻辑问题

**问题分析:**
1. `gameState.comboTimeout` 在多次游戏后可能被修改
2. 新游戏开始时未重置为配置值 `CONFIG.COMBO_TIMEOUT`
3. 导致连击检测条件判断错误，无法正常计分

**修复方案:**
```javascript
function resetGameState() {
    // ... 其他重置逻辑
    gameState.comboTimeout = CONFIG.COMBO_TIMEOUT;  // ✅ 确保每次新游戏都重置
}
```

**验证结果:** ✅ 修复后，整个 30 秒游戏过程中都能正常加分

## 🎯 游戏特性

### 核心玩法
- ⏱️ **30 秒限时挑战** - 在时间内获得最高分
- 🎯 **连击系统** - 2 秒内连续命中获得连击奖励（最高 5 倍）

- ⚠️ **倒计时警告** - 最后 5 秒红色脉冲动画

### 交互优化

- 🔇 **优雅降级** - 音效缺失时游戏仍可正常运行

## 🚀 使用说明

### 开始游戏
1. 打开 `whack-a-mole/index.html`
2. 点击"开始游戏"按钮
3. 在 30 秒内点击出现的地鼠

### 得分规则
- **基础分:** 10 分/只
- **连击奖励:** 连击倍数 × 基础分（最高 5 倍）
- **连击条件:** 2 秒内连续命中

### 操作方式
- **桌面端:** 鼠标点击地鼠
- **移动端:** 手指触摸地鼠
