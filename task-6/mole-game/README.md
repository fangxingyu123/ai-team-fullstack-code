# 🎮 打地鼠小游戏

经典的打地鼠网页小游戏，在 30 秒内尽可能多地打中地鼠！

## 🎯 游戏特色

- **3x3 网格布局**：9 个地鼠洞，地鼠随机出现
- **30 秒倒计时**：紧张刺激的时间挑战
- **连击系统**：连续击中可获得额外奖励分数
- **得分飘字**：击中时显示动态分数效果
- **锤子光标**：自定义鼠标样式，增强游戏体验
- **响应式设计**：完美支持移动端触摸操作
- **结算统计**：显示最终得分、击中次数、最大连击

## 🚀 如何开始

直接用浏览器打开 `index.html` 文件即可开始游戏！

```bash
# 或者使用本地服务器
cd mole-game
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 🎮 游戏规则

1. 点击「开始游戏」按钮
2. 地鼠会随机从洞中冒出
3. 尽快点击地鼠获得分数
4. 连续击中可触发连击奖励
5. 30 秒后游戏结束，查看你的成绩

## 📊 计分系统

- **基础得分**：每次击中 +10 分
- **连击奖励**：3 连击及以上，额外 +5 分/连击
- **连击判定**：2 秒内连续击中算作连击

## 🏆 成绩评级

- **300 分以上**：🏆 打地鼠大师！
- **200-299 分**：👏 干得漂亮！
- **100-199 分**：😊 还不错
- **100 分以下**：💪 继续练习

## 🎨 自定义

可以在 `script.js` 顶部的 `GAME_CONFIG` 中调整游戏参数：

```javascript
const GAME_CONFIG = {
    GAME_DURATION: 30,        // 游戏时长（秒）
    MOLE_APPEAR_INTERVAL: 800, // 地鼠出现间隔（毫秒）
    MOLE_STAY_DURATION: 1000,  // 地鼠停留时间（毫秒）
    BASE_SCORE: 10,           // 基础得分
    COMBO_BONUS: 5,           // 连击奖励分数
    COMBO_THRESHOLD: 3        // 触发连击的最小连续击中数
};
```
